0x00. Configurable-Middleware

This is a configurable middleware system.
This project want to solve a problem like this. In a subscribe system, every time we have to change the field subscription, we have to change the subscribe codes and re-publish the whole project. I think that it is a time waste thing, and I want to rebuild the project and make the field subscription configurable.
I used the disconf to save the class config, You can also use your DB to save the config, and regenrate the class in the codes when the subscription changes. Click here to see it on Github.
